# Vision-Gesture-Suite 

A collection of high-performance Computer Vision applications built using Python, OpenCV, and MediaPipe.

## Projects Included

### 1. AirCanvas
A gesture-driven virtual writing system. 
- **Tech:** OpenCV, MediaPipe (21 Hand Landmarks), NumPy.
- **Features:** 4-color palette selection, real-time bitwise overlays, and "hover vs. draw" finger-state logic.

### 2. Gestura
A real-time hand geometry classifier for Sign Language interpretation.
- **Tech:** Heuristic Finger-State Analysis, Spatial Coordinate Mapping.
- **Features:** Translates complex hand shapes into alphabetical characters (A, B, L, V, Y, etc.) in real-time.
